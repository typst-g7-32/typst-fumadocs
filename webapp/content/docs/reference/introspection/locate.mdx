Determines the location of an element in the document.

Takes a selector that must match exactly one element and returns that element's [`location{:typst}`](/docs/reference/introspection/location/). This location can, in particular, be used to retrieve the physical [`page{:typst}`](/docs/reference/introspection/location/#definitions-page) number and [`position{:typst}`](/docs/reference/introspection/location/#definitions-position) (page, x, y) for that element.

## Examples

Locating a specific element:

<TypstPreview
  code={`
  #context [
    Introduction is at: \
    #locate(<intro>).position()
  ]

  = Introduction <intro>
`}

```typst
#locate(
  selector
) -> location
```
