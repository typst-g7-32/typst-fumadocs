Exposes a value to the query system without producing visible content.

This element can be retrieved with the [`query{:typst}`](/docs/reference/introspection/query/) function and from the command line with [`typst query{:typst}`](/docs/reference/introspection/query/#command-line-queries). Its purpose is to expose an arbitrary value to the introspection system. To identify a metadata value among others, you can attach a [`label{:typst}`](/docs/reference/foundations/label/) to it and query for that label.

The `metadata{:typst}` element is especially useful for command line queries because it allows you to expose arbitrary values to the outside world.

<TypstPreview
  code={`
// Put metadata somewhere.
#metadata("This is a note") <note>

// And find it from anywhere else.
#context {
  query(<note>).first().value
}
`}

  alt='Preview'
/>

```typst
#metadata(
  value
) -> content
```
