Module for calculations and processing of numeric values.

These definitions are part of the `calc{:typst}` module and not imported by default. In addition to the functions listed below, the `calc{:typst}` module also defines the constants `pi{:typst}`, `tau{:typst}`, `e{:typst}`, and `inf{:typst}`.

## `round`

Rounds a number to the nearest integer.

Half-integers are rounded away from zero.

Optionally, a number of decimal places can be specified. If negative, its absolute value will indicate the amount of significant integer digits to remove before the decimal point.

Note that this function will return the same type as the operand. That is, applying `round{:typst}` to a [`float{:typst}`](/docs/reference/foundations/float/) will return a `float{:typst}`, and to a [`decimal{:typst}`](/docs/reference/foundations/decimal/), another `decimal{:typst}`. You may explicitly convert the output of this function to an integer with [`int{:typst}`](/docs/reference/foundations/int/), but note that such a conversion will error if the `float{:typst}` or `decimal{:typst}` is larger than the maximum 64-bit signed integer or smaller than the minimum integer.

In addition, this function can error if there is an attempt to round beyond the maximum or minimum integer or `decimal{:typst}`. If the number is a `float{:typst}`, such an attempt will cause `float.inf{:typst}` or `-float.inf{:typst}` to be returned for maximum and minimum respectively.

```typst
#calc.round(3.1415, digits: 2)
#assert(calc.round(3) == 3)
#assert(calc.round(3.14) == 3)
#assert(calc.round(3.5) == 4.0)
#assert(calc.round(3333.45, digits: -2) == 3300.0)
#assert(calc.round(-48953.45, digits: -3) == -49000.0)
#assert(calc.round(3333, digits: -2) == 3300)
#assert(calc.round(-48953, digits: -3) == -49000)
#assert(calc.round(decimal("-6.5")) == decimal("-7"))
#assert(calc.round(decimal("7.123456789"), digits: 6) == decimal("7.123457"))
#assert(calc.round(decimal("3333.45"), digits: -2) == decimal("3300"))
#assert(calc.round(decimal("-48953.45"), digits: -3) == decimal("-49000"))
```
