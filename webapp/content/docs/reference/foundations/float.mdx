A floating-point number.

A limited-precision representation of a real number. Typst uses 64 bits to store floats. Wherever a float is expected, you can also pass an [integer](/docs/reference/foundations/int/).

You can convert a value to a float with this type's constructor.

NaN and positive infinity are available as `float.nan{:typst}` and `float.inf{:typst}` respectively.

## Example

```typst
#3.14 \
#1e4 \
#(10 / 4)
```

## Constructor

### `float`

Converts a value to a float.

- Booleans are converted to `0.0{:typst}` or `1.0{:typst}`.
- Integers are converted to the closest 64-bit float. For integers with absolute value less than `calc.pow(2, 53){:typst}`, this conversion is exact.
- Ratios are divided by 100%.
- Strings are parsed in base 10 to the closest 64-bit float. Exponential notation is supported.

```typst
#float(false) \
#float(true) \
#float(4) \
#float(40%) \
#float("2.7") \
#float("1e5")
```

## Methods

### `is-nan`

Checks if a float is not a number.

In IEEE 754, more than one bit pattern represents a NaN. This function returns `true{:typst}` if the float is any of those bit patterns.

```typst
#float.is-nan(0) \
#float.is-nan(1) \
#float.is-nan(float.nan)
```

```#float.is-nan() -> bool```

### `is-infinite`

Checks if a float is infinite.

Floats can represent positive infinity and negative infinity. This function returns `true{:typst}` if the float is an infinity.

```typst
#float.is-infinite(0) \
#float.is-infinite(1) \
#float.is-infinite(float.inf)
```

```#float.is-infinite() -> bool```

### `signum`

Calculates the sign of a floating point number.

- If the number is positive (including `+0.0{:typst}`), returns `1.0{:typst}`.
- If the number is negative (including `-0.0{:typst}`), returns `-1.0{:typst}`.
- If the number is NaN, returns `float.nan{:typst}`.

```typst
#(5.0).signum() \
#(-5.0).signum() \
#(0.0).signum() \
#float.nan.signum()
```

```#float.signum() -> float```

### `from-bytes`

Interprets bytes as a float.

```typst
#float.from-bytes(bytes((0, 0, 0, 0, 0, 0, 240, 63))) \
#float.from-bytes(bytes((63, 240, 0, 0, 0, 0, 0, 0)), endian: "big")
```
