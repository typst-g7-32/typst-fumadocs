Evaluates a string as Typst code.

This function should only be used as a last resort.

## Example

<TypstPreview
  code={`
  #eval("1 + 1") \
  #eval("(1, 2, 3, 4)").len() \
  #eval("*Markup!*", mode: "markup") \
`}
  image='/assets/2997ea0d9ffb5751252b8ba6f78bef8f.png'
  alt='Preview'
  editable={true}
/>

```typst
#eval(
  source,
  mode: str,
  scope: dictionary
) -> any
```
