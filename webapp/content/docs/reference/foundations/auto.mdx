---
title: "Auto"
description: "Documentation for the Auto type."
---

A value that indicates a smart default.

The auto type has exactly one value: `auto{:typst}`.

Parameters that support the `auto{:typst}` value have some smart default or contextual behaviour. A good example is the [text direction](/docs/reference/text/text/#parameters-dir) parameter. Setting it to `auto{:typst}` lets Typst automatically determine the direction from the [text language](/docs/reference/text/text/#parameters-lang).
