Fails with an error.

Arguments are displayed to the user (not rendered in the document) as strings, converting with `repr{:typst}` if necessary.

## Example

The code below produces the error `panicked with: "this is wrong"{:typst}`.

```typst
#panic("this is wrong")
```

```typst
#panic(
  values
) -> 
```
