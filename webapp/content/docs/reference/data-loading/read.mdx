Reads plain text or data from a file.

By default, the file will be read as UTF-8 and returned as a [string](/docs/reference/foundations/str/).

If you specify `encoding: none{:typst}`, this returns raw [bytes](/docs/reference/foundations/bytes/) instead.

## Example

```typst
#read(
  path,
  encoding: none | str
) -> str bytes
```
