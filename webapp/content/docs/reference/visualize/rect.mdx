## `rect`

A rectangle with optional content.

## Example

```typst
// Without content.
#rect(width: 35%, height: 30pt)

// With content.
#rect[
  Automatically sized \
  to fit the content.
]
```

```#rect(width:, height:, fill:, stroke:, radius:, inset:, outset:, body) -> content```
