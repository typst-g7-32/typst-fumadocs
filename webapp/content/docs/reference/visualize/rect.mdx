A rectangle with optional content.

## Example

<TypstPreview
  code={`
  // Without content.
  #rect(width: 35%, height: 30pt)

  // With content.
  #rect[
    Automatically sized \
    to fit the content.
  ]
`}
  image='/typst-fumadocs/assets/b8c2e4acab3c02639ef4bfaa5380982a'
  alt='Preview'
  editable={true}
/>

```typst
#rect(
  width: auto | relative,
  height: auto | relative | fraction,
  fill: none | color | gradient | tiling,
  stroke: none | auto | length | color | gradient | stroke | tiling | dictionary,
  radius: relative | dictionary,
  inset: relative | dictionary,
  outset: relative | dictionary,
  body
) -> content
```
