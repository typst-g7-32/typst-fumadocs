## `polygon`

A closed polygon.

The polygon is defined by its corner points and is closed automatically.

## Example

```typst
#polygon(
  fill: blue.lighten(80%),
  stroke: blue,
  (20%, 0pt),
  (60%, 0pt),
  (80%, 2cm),
  (0%,  2cm),
)
```

### Definitions

### `regular`

A regular polygon, defined by its size and number of vertices.

```typst
#polygon.regular(
  fill: blue.lighten(80%),
  stroke: blue,
  size: 30pt,
  vertices: 3,
)
```
