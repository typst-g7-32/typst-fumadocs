A closed polygon.

The polygon is defined by its corner points and is closed automatically.

## Example

<TypstPreview
  code={`
  #polygon(
    fill: blue.lighten(80%),
    stroke: blue,
    (20%, 0pt),
    (60%, 0pt),
    (80%, 2cm),
    (0%,  2cm),
  )
`}
  image='/typst-fumadocs/assets/4eecc04e899aad583ed0d99456edd014'
  alt='Preview'
  editable={true}
/>

```typst
#polygon(
  fill: none | color | gradient | tiling,
  fill-rule: str,
  stroke: none | auto | length | color | gradient | stroke | tiling | dictionary,
  vertices
) -> content
```

### Definitions
