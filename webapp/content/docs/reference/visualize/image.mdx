## `image`

A raster or vector graphic.

You can wrap the image in a [`figure{:typst}`](/docs/reference/model/figure/) to give it a number and caption.

Like most elements, images are _block-level_ by default and thus do not integrate themselves into adjacent paragraphs. To force an image to become inline, put it into a [`box{:typst}`](/docs/reference/layout/box/).

## Example

```typst
#figure(
  image("molecular.jpg", width: 80%),
  caption: [
    A step in the molecular testing
    pipeline of our lab.
  ],
)
```
