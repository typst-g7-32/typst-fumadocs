A square with optional content.

## Example

<TypstPreview
  code={`
// Without content.
#square(size: 40pt)

// With content.
#square[
  Automatically 
  sized to fit.
]
`}
  image='/assets/e35a80a6686ae7fe68882233aa8efee.png'
  alt='Preview'
/>

```typst
#square(
  size: auto | length,
  width: auto | relative,
  height: auto | relative | fraction,
  fill: none | color | gradient | tiling,
  stroke: none | auto | length | color | gradient | stroke | tiling | dictionary,
  radius: relative | dictionary,
  inset: relative | dictionary,
  outset: relative | dictionary,
  body
) -> content
```
