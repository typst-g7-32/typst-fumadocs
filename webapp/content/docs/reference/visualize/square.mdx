## `square`

A square with optional content.

## Example

```typst
// Without content.
#square(size: 40pt)

// With content.
#square[
  Automatically \
  sized to fit.
]
```

```#square(size:, width:, height:, fill:, stroke:, radius:, inset:, outset:, body) -> content```
