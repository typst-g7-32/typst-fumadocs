A path through a list of points, connected by Bézier curves.

## Example

<TypstPreview
  code={`
#path(
  fill: blue.lighten(80%),
  stroke: blue,
  closed: true,
  (0pt, 50pt),
  (100%, 50pt),
  ((50%, 0pt), (40pt, 0pt)),
)
`}
  image='/assets/7c71fff7477a30492c8c5421fe00a40f.png'
  alt='Preview'
/>

```typst
#path(
  fill: none | color | gradient | tiling,
  fill-rule: str,
  stroke: none | auto | length | color | gradient | stroke | tiling | dictionary,
  closed: bool,
  vertices
) -> content
```
