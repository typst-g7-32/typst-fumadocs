Эллипс с необязательным содержимым.

## Пример

<TypstPreview
  code={`
  // Без содержимого.
  #ellipse(width: 35%, height: 30pt)

  // С содержимым.
  #ellipse[
    #set align(center)
    Автоматически подгоняется \\
    под размер содержимого.
  ]
`}

#ellipse(
