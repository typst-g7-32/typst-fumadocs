## `ellipse`

An ellipse with optional content.

## Example

```typst
// Without content.
#ellipse(width: 35%, height: 30pt)

// With content.
#ellipse[
  #set align(center)
  Automatically sized \
  to fit the content.
]
```

```#ellipse(width:, height:, fill:, stroke:, inset:, outset:, body) -> content```
