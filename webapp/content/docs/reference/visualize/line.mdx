Горизонтальная или вертикальная линия.

## Пример

<TypstPreview
  code={`
  #set page(height: 100pt)
  #line(length: 100%)
  Это продолжается на \\
  следующей строке.
  #line(length: 100%, stroke: 0.5pt)
`}

```typst
#line(
  start: coordinate,
  end: none | coordinate,
  length: relative,
  angle: angle,
  stroke: length | color | gradient | stroke | tiling | dictionary
) -> content
```
