A line from one point to another.

## Example

<TypstPreview
  code={`
  #set page(height: 100pt)

  #line(length: 100%)
  #line(end: (50%, 50%))
  #line(
    length: 4cm,
    stroke: 2pt + maroon,
  )
`}
  image='/typst-fumadocs/assets/20174b08a5b487d90d5ace96ffc0ca03.png'
  alt='Preview'

```typst
#line(
  start: array,
  end: none | array,
  length: relative,
  angle: angle,
  stroke: length | color | gradient | stroke | tiling | dictionary
) -> content
```
