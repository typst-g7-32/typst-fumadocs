## `line`

A line from one point to another.

## Example

```typst
#set page(height: 100pt)

#line(length: 100%)
#line(end: (50%, 50%))
#line(
  length: 4cm,
  stroke: 2pt + maroon,
)
```

```#line(start:, end:, length:, angle:, stroke:) -> content```
