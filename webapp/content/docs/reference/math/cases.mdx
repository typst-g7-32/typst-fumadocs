## `cases`

A case distinction.

Content across different branches can be aligned with the `&{:typst}` symbol.

## Example

```typst
$ f(x, y) := cases(
  1 "if" (x dot y)/2 <= 0,
  2 "if" x "is even",
  3 "if" x in NN,
  4 "else",
) $
```
