A case distinction.

Content across different branches can be aligned with the `&{:typst}` symbol.

## Example

<TypstPreview
  code={`
  $ f(x, y) := cases(
    1 "if" (x dot y)/2 <= 0,
    2 "if" x "is even",
    3 "if" x in NN,
    4 "else",
  ) $
`}
  image='/typst-fumadocs/assets/d17d4014f0e278177d8e26b02a9734f8'
  alt='Preview'
  editable={true}
/>

```typst
#math.cases(
  delim: none | str | array | symbol,
  reverse: bool,
  gap: relative,
  children
) -> content
```
