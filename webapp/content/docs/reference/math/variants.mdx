These functions are distinct from the [`text{:typst}`](/docs/reference/text/text/) function because math fonts contain multiple variants of each letter.

## `bb`

Blackboard bold (double-struck) font style in math.

For uppercase latin letters, blackboard bold is additionally available through [symbols](/docs/reference/symbols/sym/) of the form `NN{:typst}` and `RR{:typst}`.

```typst
$ bb(b) $
$ bb(N) = NN $
$ f: NN -> RR $
```

## `scr`

Script (roundhand) font style in math.

```typst
$scr(L)$ is not the set of linear
maps $cal(L)$.
```

There are two ways that fonts can support differentiating `cal{:typst}` and `scr{:typst}`. The first is using Unicode variation sequences. This works out of the box in Typst, however only a few math fonts currently support this.

The other way is using [font features](/docs/reference/text/text/#parameters-features). For example, the roundhand style might be available in a font through the _stylistic set 1_ (`ss01{:typst}`) feature. To use it in Typst, you could then define your own version of `scr{:typst}` like in the example below.

```typst
#let scr(it) = text(
  stylistic-set: 1,
  $cal(it)$,
)

We establish $cal(P) != scr(P)$.
```
