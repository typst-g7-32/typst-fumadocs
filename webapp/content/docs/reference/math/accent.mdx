Attaches an accent to a base.

## Example

<TypstPreview
  code={`
  $grave(a) = accent(a, \`)$ \
  $arrow(a) = accent(a, arrow)$ \
  $tilde(a) = accent(a, \u{0303})$
`}
  image='/typst-fumadocs/assets/c1d2d9103d9cbed5ca014ef9bcab402b.png'
  alt='Preview'
  editable={true}
/>

```typst
#math.accent(
  base,
  accent,
  size: relative,
  dotless: bool
) -> content
```
