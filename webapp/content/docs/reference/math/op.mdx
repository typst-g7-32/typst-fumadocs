## Predefined Operators

Typst predefines the operators `arccos{:typst}`, `arcsin{:typst}`, `arctan{:typst}`, `arg{:typst}`, `cos{:typst}`, `cosh{:typst}`, `cot{:typst}`, `coth{:typst}`, `csc{:typst}`, `csch{:typst}`, `ctg{:typst}`, `deg{:typst}`, `det{:typst}`, `dim{:typst}`, `exp{:typst}`, `gcd{:typst}`, `lcm{:typst}`, `hom{:typst}`, `id{:typst}`, `im{:typst}`, `inf{:typst}`, `ker{:typst}`, `lg{:typst}`, `lim{:typst}`, `liminf{:typst}`, `limsup{:typst}`, `ln{:typst}`, `log{:typst}`, `max{:typst}`, `min{:typst}`, `mod{:typst}`, `Pr{:typst}`, `sec{:typst}`, `sech{:typst}`, `sin{:typst}`, `sinc{:typst}`, `sinh{:typst}`, `sup{:typst}`, `tan{:typst}`, `tanh{:typst}`, `tg{:typst}` and `tr{:typst}`.

```typst
#math.op(
  text,
  limits: bool
) -> content
```
