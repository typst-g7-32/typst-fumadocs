Alternate letterforms within formulas.

These functions are distinct from the [`text{:typst}`](/docs/reference/text/text/) function because math fonts contain multiple variants of each letter.

Upright (non-italic) font style in math.

<TypstPreview
  code={`
  $ upright(A) != A $
`}
  image='/typst-fumadocs/assets/2375f3944b651050f90b0f7ab2b4b59e'
  alt='Preview'
  editable={true}
/>

```typst
#math.upright(
  body
) -> content
```
