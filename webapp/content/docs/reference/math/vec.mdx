## `vec`

A column vector.

Content in the vector's elements can be aligned with the [`align{:typst}`](/docs/reference/math/vec/#parameters-align) parameter, or the `&{:typst}` symbol.

This function is for typesetting vector components. To typeset a symbol that represents a vector, [`arrow{:typst}`](/docs/reference/math/accent/) and [`bold{:typst}`](/docs/reference/math/styles/#functions-bold) are commonly used.

## Example

```typst
$ vec(a, b, c) dot vec(1, 2, 3)
    = a + 2b + 3c $
```
