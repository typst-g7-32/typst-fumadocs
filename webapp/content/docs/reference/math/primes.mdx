## `primes`

Grouped primes.

```typst
$ a'''_b = a^'''_b $
```

## Syntax

This function has dedicated syntax: use apostrophes instead of primes. They will automatically attach to the previous element, moving superscripts to the next level.

```#math.primes(count) -> content```
