## `strong`

Strongly emphasizes content by increasing the font weight.

Increases the current font weight by a given `delta{:typst}`.

## Example

```typst
This is *strong.* \
This is #strong[too.] \

#show strong: set text(red)
And this is *evermore.*
```

## Syntax

This function also has dedicated syntax: To strongly emphasize content, simply enclose it in stars/asterisks (`*{:typst}`). Note that this only works at word boundaries. To strongly emphasize part of a word, you have to use the function.

```#strong(delta:, body) -> content```
