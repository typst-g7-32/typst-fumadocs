Inserts vertical spacing into a flow of blocks.

The spacing can be absolute, relative, or fractional. In the last case, the remaining space on the page is distributed among all fractional spacings according to their relative fractions.

## Example

<TypstPreview
  code={`
  #grid(
    rows: 3cm,
    columns: 6,
    gutter: 1fr,
    [A #parbreak() B],
    [A #v(0pt) B],
    [A #v(10pt) B],
    [A #v(0pt, weak: true) B],
    [A #v(40%, weak: true) B],
    [A #v(1fr) B],
  )
`}
  image='/assets/cd0b69bfd17f6ce712e61cc001c42be.png'
  alt='Preview'
  editable={true}
/>

```typst
#v(
  amount,
  weak: bool
) -> content
```
