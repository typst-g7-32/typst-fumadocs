Adds spacing around content.

The spacing can be specified for each side individually, or for all sides at once by specifying a positional argument.

## Example

<TypstPreview
  code={`
  #set align(center)

  #pad(x: 16pt, image("typing.jpg"))
  _Typing speeds can be
   measured in words per minute._
`}
  image='/typst-fumadocs/assets/627bf363796cd87adc3e0a240ccc55ab.png'
  alt='Preview'
  editable={false}
/>

```typst
#pad(
  left: relative,
  top: relative,
  right: relative,
  bottom: relative,
  x: relative,
  y: relative,
  rest: relative,
  body
) -> content
```
