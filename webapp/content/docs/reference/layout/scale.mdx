Scales content without affecting layout.

Lets you mirror content by specifying a negative scale on a single axis.

## Example

<TypstPreview
  code={`
  #set align(center)
  #scale(x: -100%)[This is mirrored.]
  #scale(x: -100%, reflow: true)[This is mirrored.]
`}
  image='/typst-fumadocs/assets/4a11fc3689aa86e118aeb7546c0a6368'
  alt='Preview'
  editable={true}
/>

```typst
#scale(
  factor: auto | length | ratio,
  x: auto | length | ratio,
  y: auto | length | ratio,
  origin: alignment,
  reflow: bool,
  body
) -> content
```
