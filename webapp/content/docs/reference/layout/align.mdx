Aligns content horizontally and vertically.

## Example

Let's start with centering our content horizontally:

<TypstPreview
  code={`
  #set page(height: 120pt)
  #set align(center)

  Centered text, a sight to see \
  In perfect balance, visually \
  Not left nor right, it stands alone \
  A work of art, a visual throne
`}
  image='/typst-fumadocs/assets/91c3481be6d803c4fd0540e78c25091a.png'
  alt='Preview'
  editable={true}
/>

To center something vertically, use _horizon_ alignment:

<TypstPreview
  code={`
  #set page(height: 120pt)
  #set align(horizon)

  Vertically centered, \
  the stage had entered, \
  a new paragraph.
`}
  image='/typst-fumadocs/assets/cbd38ef8c4834081d6b063dcffaa4d9c.png'
  alt='Preview'
  editable={true}
/>

## Combining alignments

You can combine two alignments with the `+{:typst}` operator. Let's also only apply this to one piece of content by using the function form instead of a set rule:

<TypstPreview
  code={`
  #set page(height: 120pt)
  Though left in the beginning ...

  #align(right + bottom)[
    ... they were right in the end, \
    and with addition had gotten, \
    the paragraph to the bottom!
  ]
`}
  image='/typst-fumadocs/assets/8176aa00c602f0b89c8ff5022b425216.png'
  alt='Preview'
  editable={true}
/>

## Nested alignment

You can use varying alignments for layout containers and the elements within them. This way, you can create intricate layouts:

<TypstPreview
  code={`
  #align(center, block[
    #set align(left)
    Though centered together \
    alone \
    we \
    are \
    left.
  ])
`}
  image='/typst-fumadocs/assets/7a63e59616d8948c21cd27d07c47cbd.png'
  alt='Preview'
  editable={true}
/>

## Alignment within the same line

The `align{:typst}` function performs block-level alignment and thus always interrupts the current paragraph. To have different alignment for parts of the same line, you should use [fractional spacing](/docs/reference/layout/h/) instead:

<TypstPreview
  code={`
  Start #h(1fr) End
`}
  image='/typst-fumadocs/assets/8e569fc1b13666e212c093d0351cc0de.png'
  alt='Preview'
  editable={true}
/>

```typst
#align(
  alignment,
  body
) -> content
```
