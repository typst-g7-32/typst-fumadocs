A manual page break.

Must not be used inside any containers.

## Example

<TypstPreview
  code={`
The next page contains
more details on compound theory.
#pagebreak()

== Compound Theory
In 1984, the first ...
`}
  image='/assets/3098eee9a9bf195060b49592b4463703-0.png'
  alt='Preview page 1'

Even without manual page breaks, content will be automatically paginated based on the configured page size. You can set [the page height](/docs/reference/layout/page/#parameters-height) to `auto{:typst}` to let the page grow dynamically until a manual page break occurs.

Pagination tries to avoid single lines of text at the top or bottom of a page (these are called _widows_ and _orphans_). You can adjust the [`text.costs{:typst}`](/docs/reference/text/text/#parameters-costs) parameter to disable this behavior.

```typst
#pagebreak(
  weak: bool,
  to: none | str
) -> content
```
