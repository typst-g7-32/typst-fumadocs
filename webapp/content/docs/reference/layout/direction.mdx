The four directions into which content can be laid out.

Possible values are:

- `ltr{:typst}`: Left to right.
- `rtl{:typst}`: Right to left.
- `ttb{:typst}`: Top to bottom.
- `btt{:typst}`: Bottom to top.

These values are available globally and also in the direction type's scope, so you can write either of the following two:

<TypstPreview
  code={`
  #stack(dir: rtl)[A][B][C]
  #stack(dir: direction.rtl)[A][B][C]
`}
