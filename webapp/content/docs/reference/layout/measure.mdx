## `measure`

Measures the layouted size of content.

The `measure{:typst}` function lets you determine the layouted size of content. By default an infinite space is assumed, so the measured dimensions may not necessarily match the final dimensions of the content. If you want to measure in the current layout dimensions, you can combine `measure{:typst}` and [`layout{:typst}`](/docs/reference/layout/layout/).

## Example

The same content can have a different size depending on the [context](/docs/reference/context/) that it is placed into. In the example below, the `#content{:typst}` is of course bigger when we increase the font size.

```typst
#let content = [Hello!]
#content
#set text(14pt)
#content
```

For this reason, you can only measure when context is available.

```typst
#let thing(body) = context {
  let size = measure(body)
  [Width of "#body" is #size.width]
}

#thing[Hey] \
#thing[Welcome]
```

The measure function returns a dictionary with the entries `width{:typst}` and `height{:typst}`, both of type [`length{:typst}`](/docs/reference/layout/length/).

```#measure(width:, height:, content) -> dictionary```
