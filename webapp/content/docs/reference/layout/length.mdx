A size or distance, possibly expressed with contextual units.

Typst supports the following length units:

- Points: `72pt{:typst}`
- Millimeters: `254mm{:typst}`
- Centimeters: `2.54cm{:typst}`
- Inches: `1in{:typst}`
- Relative to font size: `2.5em{:typst}`

You can multiply lengths with and divide them by integers and floats.

## Example

<TypstPreview
  code={`
#rect(width: 20pt)
#rect(width: 2em)
#rect(width: 1in)

#(3em + 5pt).em 
#(20pt).em 
#(40em + 2pt).abs 
#(5em).abs
`}

## Fields

- `abs{:typst}`: A length with just the absolute component of the current length (that is, excluding the `em{:typst}` component).
- `em{:typst}`: The amount of `em{:typst}` units in this length, as a [float](/docs/reference/foundations/float/).


## Methods
Converts this length to points.

Fails with an error if this length has non-zero `em{:typst}` units (such as `5em + 2pt{:typst}` instead of just `2pt{:typst}`). Use the `abs{:typst}` field (such as in `(5em + 2pt).abs.pt(){:typst}`) to ignore the `em{:typst}` component of the length (thus converting only its absolute component).

Converts this length to millimeters.

Fails with an error if this length has non-zero `em{:typst}` units. See the [`pt{:typst}`](/docs/reference/layout/length/#definitions-pt) method for more details.

Converts this length to centimeters.

Fails with an error if this length has non-zero `em{:typst}` units. See the [`pt{:typst}`](/docs/reference/layout/length/#definitions-pt) method for more details.

Converts this length to inches.

Fails with an error if this length has non-zero `em{:typst}` units. See the [`pt{:typst}`](/docs/reference/layout/length/#definitions-pt) method for more details.

Resolve this length to an absolute length.

<TypstPreview
  code={`
#set text(size: 12pt)
#context [
  #(6pt).to-absolute() 
  #(6pt + 10em).to-absolute() 
  #(10em).to-absolute()
]

#set text(size: 6pt)
#context [
  #(6pt).to-absolute() 
  #(6pt + 10em).to-absolute() 
  #(10em).to-absolute()
]
`}
