## `skew`

Skews content.

Skews an element in horizontal and/or vertical direction. The layout will act as if the element was not skewed unless you specify `reflow: true{:typst}`.

## Example

```typst
#skew(ax: -12deg)[
  This is some fake italic text.
]
```

```#skew(ax:, ay:, origin:, reflow:, body) -> content```
