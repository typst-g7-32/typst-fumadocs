Renders text in superscript.

The text is rendered smaller and its baseline is raised.

## Example

<TypstPreview
  code={`
1#super[st] try!
`}
  image='/assets/d39db3c0aae4bd51ed655cd6eb958575.png'
  alt='Preview'
/>

```typst
#super(
  typographic: bool,
  baseline: auto | length,
  size: auto | length,
  body
) -> content
```
