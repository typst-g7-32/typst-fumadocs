Adds a line over text.

## Example

<TypstPreview
  code={`
  #overline[A line over text.]
`}
  image='/assets/50989a8ae293089193aedd0105c6c64.png'
  alt='Preview'
  editable={true}
/>

```typst
#overline(
  stroke: auto | length | color | gradient | stroke | tiling | dictionary,
  offset: auto | length,
  extent: length,
  evade: bool,
  background: bool,
  body
) -> content
```
