Underlines text.

## Example

<TypstPreview
  code={`
This is #underline[important].
`}
  image='/assets/c55f85cbccf075521fc87c8ea5d93ff4.png'
  alt='Preview'
/>

```typst
#underline(
  stroke: auto | length | color | gradient | stroke | tiling | dictionary,
  offset: auto | length,
  extent: length,
  evade: bool,
  background: bool,
  body
) -> content
```
