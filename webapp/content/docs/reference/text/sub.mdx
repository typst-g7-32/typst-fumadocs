Renders text in subscript.

The text is rendered smaller and its baseline is lowered.

## Example

<TypstPreview
  code={`
  Revenue#sub[yearly]
`}
  image='/typst-fumadocs/assets/aba9b70776d538b28fb89148a20a8833.png'
  alt='Preview'
  editable={true}
/>

```typst
#sub(
  typographic: bool,
  baseline: auto | length,
  size: auto | length,
  body
) -> content
```
