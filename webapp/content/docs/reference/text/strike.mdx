Strikes through text.

## Example

<TypstPreview
  code={`
  This is #strike[not] relevant.
`}
  image='/typst-fumadocs/assets/8189864734cb2541923731f3119141de.png'
  alt='Preview'
  editable={true}
/>

```typst
#strike(
  stroke: auto | length | color | gradient | stroke | tiling | dictionary,
  offset: auto | length,
  extent: length,
  background: bool,
  body
) -> content
```
