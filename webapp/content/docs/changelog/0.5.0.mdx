---
title: "0.5.0"
description: "Changes in Typst 0.5.0"
---


## Text and Layout

- Added [`raw{:typst}`](/docs/reference/text/raw/) syntax highlighting for many more languages
- Added support for Korean [numbering](/docs/reference/model/numbering/)
- Added basic i18n for a few more languages (NL, SV, DA)
- Improved line breaking for East Asian languages
- Expanded functionality of outline [`indent{:typst}`](/docs/reference/model/outline/#parameters-indent) property
- Fixed footnotes in columns
- Fixed page breaking bugs with [footnotes](/docs/reference/model/footnote/)
- Fixed bug with handling of footnotes in lists, tables, and figures
- Fixed a bug with CJK punctuation adjustment
- Fixed a crash with rounded rectangles
- Fixed alignment of [`line{:typst}`](/docs/reference/visualize/line/) elements

## Math

- **Breaking change:** The syntax rules for mathematical [attachments](/docs/reference/math/attach/#functions-attach) were improved: `$f^abs(3)${:typst}` now parses as `$f^(abs(3))${:typst}` instead of `$(f^abs)(3)${:typst}`. To disambiguate, add a space: `$f^zeta (3)${:typst}`.
- Added [forced size](/docs/reference/math/sizes/) commands for math (e.g., [`display{:typst}`](/docs/reference/math/sizes/#functions-display))
- Added [`supplement{:typst}`](/docs/reference/math/equation/#parameters-supplement) parameter to [`equation{:typst}`](/docs/reference/math/equation/), used by [references](/docs/reference/model/ref/)
- New [symbols](/docs/reference/symbols/sym/): `bullet{:typst}`, `xor{:typst}`, `slash.big{:typst}`, `sigma.alt{:typst}`, `tack.r.not{:typst}`, `tack.r.short{:typst}`, `tack.r.double.not{:typst}`
- Fixed a bug with symbols in matrices
- Fixed a crash in the [`attach{:typst}`](/docs/reference/math/attach/#functions-attach) function

## Scripting

- Added new [`datetime{:typst}`](/docs/reference/foundations/datetime/) type and [`datetime.today{:typst}`](/docs/reference/foundations/datetime/#definitions-today) to retrieve the current date
- Added [`str.from-unicode{:typst}`](/docs/reference/foundations/str/#definitions-from-unicode) and [`str.to-unicode{:typst}`](/docs/reference/foundations/str/#definitions-to-unicode) functions
- Added [`fields{:typst}`](/docs/reference/foundations/content/#definitions-fields) method on content
- Added `base{:typst}` parameter to [`str{:typst}`](/docs/reference/foundations/str/) function
- Added [`calc.exp{:typst}`](/docs/reference/foundations/calc/#functions-exp) and [`calc.ln{:typst}`](/docs/reference/foundations/calc/#functions-ln)
- Improved accuracy of [`calc.pow{:typst}`](/docs/reference/foundations/calc/#functions-pow) and [`calc.log{:typst}`](/docs/reference/foundations/calc/#functions-log) for specific bases
- Fixed [removal](/docs/reference/foundations/dictionary/#definitions-remove) order for dictionary
- Fixed `.at(default: ..){:typst}` for [strings](/docs/reference/foundations/str/#definitions-at) and [content](/docs/reference/foundations/content/#definitions-at)
- Fixed field access on styled elements
- Removed deprecated `calc.mod{:typst}` function

## Command line interface

- Added PNG export via `typst compile source.typ output-\{n\}.png{:typst}`. The output path must contain `\{n\}{:typst}` if the document has multiple pages.
- Added `--diagnostic-format=short{:typst}` for Unix-style short diagnostics
- Doesn't emit color codes anymore if stderr isn't a TTY
- Now sets the correct exit when invoked with a nonexistent file
- Now ignores UTF-8 BOM in Typst files

## Miscellaneous Improvements

- Improved errors for mismatched delimiters
- Improved error message for failed length comparisons
- Fixed a bug with images not showing up in Apple Preview
- Fixed multiple bugs with the PDF outline
- Fixed citations and other searchable elements in [`hide{:typst}`](/docs/reference/layout/hide/)
- Fixed bugs with [reference supplements](/docs/reference/model/ref/#parameters-supplement)
- Fixed Nix flake
